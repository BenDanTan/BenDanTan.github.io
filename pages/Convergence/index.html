Convergence Resource Center

My second ASP.NET project, product of Code the Way 2019

Project Description

Convergence Resource Center (CRC) is a Milwaukee-based non-profit organization which takes on the issues with human trafficking. The program, however, did not have an easy, internet based way for people in need to get in contact with the organization, so Code the Way stepped in in 2019. The application we wrote was a web-based application which allowed users to contact and meet with the CRC to get help, as well as view events in the community. The application also offers two factor authentication to avoid harmful tampering and unauthorized access as this issue can quickly become very serious in the context of human trafficking. The application is currently being used by the CRC and can be found at convergenceresourcecenterapp.org

Personal Reflection

Project Reflection

The Convergence Resource Center project was an incredible experience all around. This project was the first one I had taken a leadership role in and it was for a fantastic cause so throughout the project I was determined to create a good, working project. Fortunately enough, we were able to deliver a fully functional application by the end of the summer and it was very quickly put to use. Beyond the satisfaction of creating a working product, it is incredibly rewarding to know that our application has already helped people get the help they need. This being my last project before beginning my college career, I feel as though the experience prepared me well for the continuing my software development at the university level while giving me a meaningful project to be proud of.
