File I/O

Project Description

This project was my introduction to file input/output in C#. The program took an input file of numbers and gave the user a list of possible output algorithms like outputting all even numbers or the sum of all the numbers. There was nothing particularly difficult or interesting about this project but it was a skill that I needed to be sure I had before moving on in the Code the Way season.

Code

C# file I/O code

 


using System;
    using System.Collections.Generic;
    using System.IO;
    using System.Linq;
    using System.Reflection;
    using System.Threading.Tasks;



    namespace FileIOC
    {
        public class MainClass
        {
            public static void Main(string[] args)
            {
                Console.WriteLine("please enter the name of the input file:");
                string inputFileName = Console.ReadLine();
                var inPath = Path.GetDirectoryName(Assembly.GetExecutingAssembly().Location);
                inPath += "/" + inputFileName + ".txt";
                Console.WriteLine("please enter the name of the output file:");
                string outFileName = Console.ReadLine();
                var outPath = Path.GetDirectoryName(Assembly.GetExecutingAssembly().Location);
                outPath += "/" + outFileName + ".txt";
                string[] lines = System.IO.File.ReadAllLines(inPath);
                System.IO.StreamWriter file = new System.IO.StreamWriter(outPath);
                Console.WriteLine("1-List\n2-sum\n3-even\n4-odd\n5-fifth\n6-div by three");
                string option = Console.ReadLine();
                int intTemp = Convert.ToInt32(option);
                switch (intTemp)
                {
                    case 1:
                    FileIO.Sum(lines, file);
                    break;

                    case 2:
                    FileIO.WriteAll(lines, file);
                    break;
                    case 3:
                    FileIO.Evens(lines, file);
                    break;

                    case 4:
                    FileIO.Odds(lines, file);
                    break;

                    case 5:
                    FileIO.Fifth(lines, file);
                    break;

                    case 6:
                    FileIO.DivThree(lines, file);
                    break;

                    default:

                    break;

                }

            }

        }

    }

Project Reflection

Learning about File IO in C# exposed me to a lot of new skills and concepts. This project was my first real exposure to developing in both C# and Source Control in Visual Studio. I had already done File IO in Java using GitKraken and GitHub source control through Eclipse, but converting from Java to C# was a new step I needed to take.
