Server Side Unit Conversion

Project Description

A unit conversion tool using C# and the MVC structure. Using this structure for unit conversion set up the groundwork for using the entire MVC framework and implementing a database.

Controller

Unit conversion controller written in C#

 


using System;
    using System.Collections.Generic;
    using System.Linq;
    using System.Web;
    using System.Web.Mvc;

    namespace ServerConversion.Controllers
    {
        public class FootInchesController : Controller
        {
        // GET: FootInches
        public ActionResult FootInches()
        {
            return View(viewName: "FootInches", model: 0.0);
        }
        public ActionResult Convert(double feet)
        {
            double inches = feet * 12;
            return View(viewName: "FootInches", model: inches);
            }
        }
    }

Project Reflection

Creating a Unit Conversion tool using the MVC structure was an important step in my Code the Way education. It was my first look into how MVC works and how each of the individual components work. It was also the most difficult portion of the entire summer camp program as it took nearly everything I had learned so far and complicated it as I tried to combine all of the pieces. Using all of the different skills I had learned over the last couple weeks allowed me to see how all of them work together to improve upon each other, using the strengths of each component to create the best product possible.
