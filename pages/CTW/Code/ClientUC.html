Client Side Unit Conversion

Project Description

A unit conversion page written using HTML and JavaScript. This program introduced JavaScript and data manipulation alongside HTML to the Code the Way program.

Code

Unit conversion application written in HTML with JavaScript

 


<head>
    <meta charset="utf-8" />

    <title>Feet-Inches</title>
    <link rel="stylesheet" type="text/css" href="Site.css" />
    <link href="bootstrap.css" type="text/css" rel="stylesheet" />
</head>
    <body>
        <div class="topnav">
            <a href="index.html">Feet to Meters</a>
            <a href="C to F.html">Celsius to Fahrenheit</a>
            <a href="feetInches.html">Feet to Inches</a>
            <a href="base.html">Base to Base</a>
            <a href="centimeterToMeter.html">Centimeter to Meter</a>
        </div>
        <div class="header darkBlue">
            <h1 class="header">Feet to Inches</h1>
        </div>
        <div id="app">
            <div class="row">
                <div class="offset-3 col-2 bgBlue shadow-lg p-3 mb-5 rounded">
                    <div class="col-6">
                        <label for="Number of feet" class="centerize">Feet:</label>
                        <input v-model="feet" class="input lineBoarder" type="number" />
                    </div>
                </div>
                <div class=" offset-2 col-2 bgBlue shadow-lg p-3 mb-5 rounded">
                    <div class="col-6">
                        <label for="Number of inches">Inches:</label>
                        <div class="output lineBoarder">
                            <p>{{ inches }}</p>
                        </div>
                    </div>
                </div>
            </div>
            <div class="centerize">
                <button v-on:click="feetInches" class="buttonB medBlue">Convert</button>
            </div>
        </div>
    </body>
                
<script src="https://unpkg.com/vue"></script>
    <script>
        var app = new Vue({
            el: '#app',
            data: {
            feet: "",
            inches: ""
        },
        methods: {
            feetInches: function () {
                this.inches = this.feet * 12;
            }
        }
    })
</script>

Project Reflection

Creating a unit conversion tool in HTML and JavaScript was particularly interesting as it was my first exposure to JavaScript. I had little trouble with this skill, however, as it was taught thoroughly and my team came together to work through any roadblocks that popped up. This section was also a much needed review of HTML and improved my HTML skills with the integration of JavaScript. All-in-all this was a very meaningful and helpful activity to improve my skill set and introduce me to JavaScript and different elements of HTML.
