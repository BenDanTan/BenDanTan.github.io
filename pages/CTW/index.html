Code the Way Bootcamp

Learning and using ASP.NET and MVC for the first time

Project Description

Before we were able to start working on projects in the Code the Way program we needed to learn a lot. Most of us in the program had only really written code in Java, so moving to full stack ASP.NET development was going to be a significant jump. Given the short, nine week season we had for the entire Code the Way program, only three weeks were allocated to the educational portion of the program. In this time we would need to learn Scrum project management, File IO, data manipulation on both the client and server side, MVC and ASP.NET as well as look at all of the languages we would be using in the project. Learn more about Code the Way from Code the Way, Safenet Consulting or Elmbrook School District.

Personal Reflection

Project Reflection

The Summer Camp program of 2018 was certainly an experiment, this was clear in many instances. But despite the many problems we encountered, I learned a lot as, I am sure, everybody else did. The program introduced me to several new concepts and skills in a very short amount of time and while it was stressful, I found myself enjoying the steep learning curve and benefiting from the fast pace at which the course ran. The biggest issue with the program ended up being the different types of machines we had running. Windows based computers generally worked fine, while Macbooks ran into countless errors and even threw the rest of us off. This is where the experimental nature of the program really showed, when the entire program would halt to wait for somebody to try and whip up a solution to whatever problem the Mac users were having. Overall, however, despite the hiccups I believe the program was extremely beneficial to everybody that was a part of it, and it was run extremely well, especially considering this was the first time the program had run.
